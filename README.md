**Fighting Game - Simulação de Luta**

Este é um projeto simples de uma simulação de luta, desenvolvido para demonstrar os princípios da Programação Orientada a Objetos (POO) e a manipulação do DOM com JavaScript puro.

Tecnologias Utilizadas
HTML: Estrutura básica da página.

CSS: Estilização da interface.

JavaScript: Lógica principal do jogo, manipulação de objetos e eventos.

O projeto é focado em JavaScript, que representa mais de 60% do código, utilizando funções e classes para criar e gerenciar os lutadores.

Como Executar o Projeto
Para visualizar a simulação de luta, siga estes passos simples:

Clone o repositório:

Bash

git clone https://github.com/mateushoffmann7/-object-oriented-project-fight

Abra o arquivo index.html em seu navegador.

Se preferir, use uma extensão do seu editor de código, como o Live Server, para rodar o projeto em um servidor local.

Funcionalidades
Personagens: Arthas e Hadggar com atributos de vida e ataque.

Ataque: Cada personagem pode atacar o outro com um sistema de dano baseado em fatores aleatórios de ataque e defesa.

Interface: Barras de vida que se atualizam em tempo real para mostrar o dano sofrido.

Estrutura do Código
O código é dividido em arquivos para organização:

index.html: O arquivo principal que carrega os scripts.

style.css: O arquivo de estilos.

script.js: Contém a classe principal para os personagens e a lógica do jogo.

functions.js: Contém funções utilitárias que podem ser compartilhadas.

Créditos
Desenvolvido como um projeto de estudo de Programação Orientada a Objetos (POO) com JavaScript.
